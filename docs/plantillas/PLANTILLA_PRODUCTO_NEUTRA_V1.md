# PLANTILLA_PRODUCTO_NEUTRA_V1

Estado: APROBADA

## Uso obligatorio
Plantilla oficial para:
- Medicamentos
- Tests
- Bacterias
- Aditivos
- Alimentos
- Sales
- Equipamiento pequeño

## Fondo fijo
- Mesa o estantería neutra.
- Pared clara y limpia.
- Ambiente realista.
- Planta decorativa opcional.
- Estilo catálogo premium.

## Protagonista
- El producto es el protagonista absoluto.
- El envase debe verse completo y reconocible.
- La fotografía real del producto tiene prioridad.

## Tipografía fija
- Nombre principal grande.
- Estilo premium coherente con NexoCreator.
- Subtítulo debajo.
- Información resumida en la zona inferior.

## Composición fija
- Producto centrado.
- Nombre del producto debajo.
- Subtítulo debajo del nombre.
- Iconos o puntos clave en la parte inferior.
- No usar el texto 'Producto Premium'.

## Variables
- {{imagen_principal}}
- {{nombre_producto}}
- {{subtitulo}}
- {{beneficios_clave}}

## Regla maestra
La misma plantilla sirve para marino y dulce. No crear versiones separadas por tipo de agua.
