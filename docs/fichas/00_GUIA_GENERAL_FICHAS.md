# Guía general de fichas

## Objetivo
Crear fichas consistentes, verificables y exportables.

## Flujo
1. Foto o datos base.
2. Identificación.
3. Verificación de información.
4. Creación visual.
5. Exportación JSON.

## Campos mínimos
- Nombre común
- Nombre científico
- Categoría
- Portada
- Descripción
- Fuente revisada

## Reglas
- No inventar datos.
- Marcar lo no verificado.
- Conservar la portada en el JSON mediante cover_image.
- Mantener estructura compatible con AcuarioNexo.
