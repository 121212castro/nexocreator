# PLANTILLA_DULCE_PLANTADO_V1

Estado: APROBADA

## Propósito
Portada oficial para peces de comunitario plantado, gambas de agua dulce, otocinclus, rasboras, neones y especies que encajan mejor en un acuario verde, limpio y moderno.

## Fondo fijo
- Acuario plantado premium.
- Verde intenso y luminoso.
- Plantas acuáticas densas.
- Rocas y troncos integrados, no protagonistas.
- Agua clara con partículas o burbujas suaves.
- Zona central despejada para el protagonista.
- Aspecto moderno, limpio y natural.

## Tipografía fija
- Mismo estilo que las plantillas dulces aprobadas.
- Nombre científico grande.
- Serif elegante en cursiva.
- Color blanco cálido o marfil.
- Nombre común debajo.
- Color dorado cálido.

## Composición fija
- El protagonista es siempre el pez, gamba o especie principal.
- El texto se coloca en la parte inferior.
- El texto nunca debe competir con el protagonista.
- Separador decorativo fino.
- Mantener zona central libre para la imagen principal.

## Primera referencia validada
- Acuario plantado
- Plantado premium

## Regla
Toda ficha que use esta plantilla debe conservar fondo, tipografía y composición. Solo cambian las variables.
