# PLANTILLA_MARINA_PREMIUM_V1

Estado: APROBADA

## Uso obligatorio
Esta es la plantilla visual oficial para toda la biblioteca marina.

Se utiliza para:

- Peces marinos
- Corales
- Invertebrados marinos

## Fondo fijo
- Reef azul profundo.
- Rayos de luz desde la superficie.
- Rocas y estructura de arrecife.
- Sensación de profundidad marina.
- Zona central libre para el protagonista.

## Tipografía fija
- Nombre científico grande.
- Serif elegante en cursiva.
- Color marfil/blanco cálido.
- Nombre común debajo.
- Dorado cálido.

## Composición fija
- El protagonista siempre domina visualmente la portada.
- El texto se sitúa en la parte inferior.
- El texto nunca compite con el protagonista.
- Separador decorativo fino.

## Variables
- {{imagen_principal}}
- {{nombre_cientifico}}
- {{nombre_comun}}

## Regla maestra
Peces marinos, corales e invertebrados marinos comparten exactamente la misma identidad visual.

Solo cambian las variables.

No crear variantes de coral ni variantes de invertebrado mientras esta plantilla siga aprobada.
