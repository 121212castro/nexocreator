# REGLA_MICROFAUNA_MARINA_V1

Estado: APROBADA

## Regla maestra
La microfauna marina utiliza la plantilla marina oficial.

Microfauna marina
→ PLANTILLA_MARINA_PREMIUM_V1

## Aplica a
- Copépodos marinos
- Rotíferos marinos
- Artemia para marino
- Fitoplancton marino
- Nannochloropsis
- Tetraselmis
- Isochrysis

## Variables
- imagen_principal
- nombre_cientifico
- nombre_comun

No crear una plantilla específica de microfauna marina mientras esta regla siga aprobada.
