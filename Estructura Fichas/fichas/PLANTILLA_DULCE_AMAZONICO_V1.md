# PLANTILLA_DULCE_AMAZONICO_V1

Estado: APROBADA

## Propósito
Portada oficial para peces amazónicos y especies de agua dulce asociadas a ambientes de raíces, troncos y vegetación natural.

## Fondo fijo
- Río amazónico sumergido.
- Tonos verdes naturales.
- Rayos de luz descendentes desde la superficie.
- Troncos y raíces laterales.
- Zona central despejada para el protagonista.
- Sensación de profundidad.

## Tipografía fija
- Nombre científico grande.
- Serif elegante en cursiva.
- Color blanco cálido o marfil.
- Nombre común debajo.
- Color dorado cálido.

## Composición fija
- El protagonista es siempre el pez.
- El pez ocupa la zona central visual.
- El texto se sitúa en la parte inferior.
- El texto nunca debe competir visualmente con el pez.
- Separador decorativo fino.
