# PLANTILLA_PRODUCTO_CLINICO_V1

Estado: APROBADA

## Uso obligatorio
Esta plantilla se utiliza para:

- Medicamentos
- Bacterias
- Acondicionadores
- Tests
- Aditivos
- Tratamientos
- Productos de acuariofilia

## Fondo fijo
- Fondo técnico oscuro.
- Azul profundo o grafito.
- Iluminación controlada.
- Aspecto profesional.
- Ambiente limpio.
- Sin elementos naturales dominantes.

## Protagonista
- El producto es el protagonista absoluto.
- El envase debe verse claramente.
- La etiqueta debe ser reconocible.

## Tipografía fija
- Misma familia visual premium de NexoCreator.
- Nombre comercial principal grande.
- Subtítulo descriptivo debajo.
- Composición limpia y legible.

## Composición fija
- Producto centrado.
- Texto en zona inferior.
- Fondo sin distracciones.
- Aspecto catálogo premium.

## Variables
- {{imagen_principal}}
- {{nombre_producto}}
- {{subtitulo}}

## Regla
El usuario debe identificar el producto de un vistazo. El producto siempre domina visualmente la portada.
